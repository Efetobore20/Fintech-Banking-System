# Fintech Banking System

This is a simple fintech banking system developed using PHP and MySQL. 
The project was built in a local development environment using XAMPP.

## Features
- User registration
- User login authentication
- Database storage using MySQL
- Transaction processing
- Responsive design with Bootstrap

## Technologies Used
- PHP
- MySQL
- HTML
- CSS
- Bootstrap
- JavaScript

## How to Run the Project
1. Install XAMPP.
2. Place the project folder inside the htdocs directory.
3. Import the database.sql file into phpMyAdmin.
4. Start Apache and MySQL in XAMPP.
5. Open the project in the browser using localhost.
